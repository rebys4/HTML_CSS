# HTML_CSS
Конспект по курсу HTML/CSS/JS

## Краткая информация про HTML
**HTML** - это язык разметки, который используется для создания веб-страниц. Он представляет собой способ **описания** структуры и содержания веб-страницы, **позволяя** браузерам интерпретировать и отображать:
1) Текст;
2) Изображения;
3) Ссылки;
4) Видео и другие элементы на веб-сайте.

HTML **использует** специальные **теги** (кодовые метки в угловых скобках <>), чтобы определить, какой текст должен быть заголовком, параграфом, изображением, ссылкой и т. д. Эти теги **позволяют** создавать структуру страницы, делая её более понятной для браузеров и поисковых систем.

## HTML: Теги
Пример **двойного тега**:
```
<p>
    Привет, мир!
</p>
```
В данном примере двойной тег - это ```<p>```. Двойные теги показывают начало и конец элемента. 
>[!WARNING]
> - **Начало** элемента обозначают - открывающимся тегом <...>
> - **Конец** элемента - закрывающим </...>
> - **Текст "Привет мир" в данном случае является контентом(Content)**. Это контент элемента, который в данном случае является просто текстом.
> - **Элемент (Element)**: Открывающий тег, закрывающий тег и контент вместе составляют элемент

**Помимо двойных тегов существуют одинарные, и они не имеют пары.**

### HTML: Атрибуты
**Атрибуты** - это дополнительная **информация**, которая присваивается HTML-элементам, чтобы **определить** их свойства или параметры.
>[!WARNING]
> Они **указываются внутри открывающего** тега элемента и предоставляют браузеру или другим программам дополнительные инструкции о том, как элемент должен вести себя или как его следует отображать на веб-странице.

**Синтаксис атрибута**:
```HTML
<элемент атрибут="значение">
```
- ```<элемент>``` - это HTML-элемент, к которому применяется атрибут.
- ```атрибут``` - имя атрибута, указывающее, какое свойство элемента будет изменено.
- ```значение``` - значение атрибута, которое определяет, каким образом будет изменено свойство элемента.


## HTML: Заголовки
**Заголовки** являются важной частью **HTML** документа, обычно **заголовок** — это краткое описание того, что будет представлено ниже на странице

>[!Warning]
>Заголовки в HTML создаются с помощью тега ```h``` после которого идет цифра от 1 до 6, например ```<h1>```, где:
> - ```h``` - сокращение от английского heading (заголовок);
> - 1..6 - уровни заголовка;
> - Все теги являются парными.

**Верхним** уровнем является - 1 (```<h1>```), самым **нижним** - уровень 6 (```<h6>```).

### HTML: Параграфы
**Параграф** - это один из **основных** HTML-элементов, используемых для оформления и структурирования текста на веб-странице. Элемент ```<p>``` обозначает абзац текста и позволяет создавать разделение между разными абзацами контента.

**Синтаксис** тега параграфа:
```HTML
<p>
    Ваш текст здесь
</p>
```
- ```<p>``` - это **открывающий** тег параграфа.
- ```</p>``` - это **закрывающий** тег параграфа.

>[!WARNING]
> Текст, который находится **между** этими тегами, будет рассматриваться как отдельный абзац и отображаться соответствующим образом на веб-странице.

### HTML: Списки
**Список** - это набор взаимосвязанных элементов, которые используются для группировки какой либо информации.

**Списки** бывают следующих видов:
- Нумерованный
- Маркированный
- Вложенный
- Список определений

*Разберем каждый подробнее*

### **Нумерованный**

В **нумерованном** списке перед каждым элементом списка идет **номер**.

Он **создается** при помощи тега ```<ol>``` — ordered list (упорядоченный список). Каждый пункт списка создаётся с помощью элемента ```<li>``` — list item (пункт списка).

### **Маркированный**

В **маркированном** списке перед каждым элементом списка идет маркер.

Маркированный список **создается** с помощью тега ```<ul>``` — unordered list (неупорядоченный список). Стандартный маркер — точка. Каждый пункт списка также создаётся с помощью элемента ```<li>``` — list item (пункт списка).

### **Вложенный**
У нас на очереди **вложенный** список, с помощью **вложенных** списков можно вкладывать одни элементы в другие, тем самым у нас получаются подпункты.

Чтобы создать **вложенный** список в HTML нужно использовать все те же элементы. Суть проста, в элемент ```<li>...</li>``` вкладывается новый маркированный или нумерованный список, начиная с элемента ```<ul>``` или ```<ol>```.

### **Список определений**
**Список определений** (Definition List) - это структура в HTML, которая используется для **представления** определений или терминов вместе с их описаниями.

>[!WARNING]
> Список определений состоит из пар "термин - описание" и предоставляет удобный способ организации информации.

Он использует следующие теги:

1) ```<dl>``` **(Definition List)**: Этот тег представляет собой обертку для всего списка определений. Он обрамляет все пары "термин - описание".
2) ```<dt>``` **(Definition Term)**: Этот тег используется для определения термина или того, что вы хотите определить. Он идет внутри тега ```<dl>```.
3) ```<dd>``` **(Definition Description)**: Этот тег используется для предоставления описания или определения для соответствующего термина, который находится в теге ```<dt>```. Тег ```<dd>``` также идет внутри тега ```<dl>```.

### HTML: Тег переноса
Перенос строк текста в HTML через тег ```<br/>```

```<br/>``` этот тег указывающий на место разрыва строки и принудительного переноса текста на новую строку. И только для этих целей он должен применяться.

### HTML: Блочные теги

**Блочные теги** (или блочные элементы) - это одна из двух **основных** категорий HTML-элементов, которые используются для организации и структурирования содержимого на веб-страницах.

Блочные элементы **характеризуются** следующими особенностями:

1) **Основная черта блочных элементов** - они **занимают всю доступную горизонтальную ширину на странице** и начинаются с **новой** строки после себя. Это означает, что каждый блочный элемент начинается с новой строки и занимает всю доступную горизонтальную ширину на странице, что делает их подходящими для организации больших блоков контента.
2) **Примеры блочных элементов**: Некоторые популярные блочные элементы в HTML включают ```<div>, <p>, <h1> до <h6>, <ul>, <ol>, <li>, <table>, <form>``` и многие другие.
3) **Вложенность**: Блочные элементы могут **вкладываться** друг в друга, что позволяет создавать сложные структуры и макеты веб-страниц.

## СТРОЧНЫЕ ТЕГИ ДЛЯ РАБОТЫ С ТЕКСТОМ
- ```<b>``` - Определяет жирное начертание шрифта.
- ```<strong>``` - Этот тег делает текст жирным и обычно используется для выделения важных слов или фраз.
- ```<em>``` - Этот тег придает тексту курсивное начертание и используется для выделения текста с эмоциональным акцентом или подчеркивания важности.
- ```<del>``` - Этот тег перечеркивает текст и используется для обозначения удаленных или устаревших частей текста.
- ```<abbr>``` - Этот тег используется для обозначения аббревиатур, и при наведении курсора показывает расшифровку аббревиатуры.
- ```<mark>``` - Этот тег используется для выделения текста ярким цветом, обычно для подчеркивания важных фрагментов текста.
- ```<i>``` - Устанавливает курсивное начертание шрифта.
- ```<u>``` - Этот тег подчеркивает текст и может использоваться для обозначения подчеркнутых слов или фраз.
- ```<span>``` - Универсальный тег, предназначенный для определения строчного элемента внутри документа.
- ```<sub>``` - Отображает шрифт в виде нижнего индекса. Текст при этом располагается ниже базовой линии остальных символов строки и уменьшенного размера — H<sub>2</sub>O.
- ```<sup>``` - Отображает шрифт в виде верхнего индекса. По своему действию похож на ```<sub>```, но текст отображается выше базовой линии текста — м<sup>2</sup>.
- ```<big>``` - Увеличенный текст.
- ```<small>``` - Уменьшенный текст.


## HTML: АТРИБУТЫ 

### HTML: Документ и его структура

**HTML** (HyperText Markup Language) **документ** имеет четкую **структуру**, которая состоит из нескольких ключевых элементов. 

Вот основная **структура** HTML документа:
```HTML
<!DOCTYPE html> <!-- Объявление формата документа -->
<html>
    <head> <!-- Техническая информация о документе -->
        <meta charset="UTF-8"> <!-- Определяем кодировку символов документа -->
        <title>...</title> <!-- Задаем заголовок документа -->
    </head>
    <body> 
        <!-- Основная часть документа -->
    </body>
</html>
```

1) ```<!DOCTYPE>``` объявление
Первой строкой в HTML документе обычно идет объявление типа документа (<!DOCTYPE>).
>[!NOTE]
> Он сообщает браузеру, какой стандарт HTML используется в документе.

2) ```<html>```
Элемент ```<html>``` представляет начало и конец всего HTML документа и обертывает всё содержимое документа.
3) ```<head>```
Элемент ```<head>``` содержит **метаданные** документа, такие как заголовок страницы, ссылки на внешние ресурсы (CSS, JavaScript), мета-теги и другую информацию, которая не отображается напрямую на странице

Пример:
```HTML
<head> 
    <title>Заголовок страницы</title> 
    <link rel="stylesheet" type="text/css" href="styles.css"> 
    <!-- Другие мета-теги и ссылки --> 
</head>
```
4) ```<title>```
Элемент ```<title>``` задает заголовок страницы, который отображается во вкладке браузера или при сохранении закладки.
5) ```<meta>```
Элемент ```<meta>``` используется для определения метаданных, таких как кодировка страницы и описание документа.
6) ```<body>```
Элемент ```<body>``` содержит основное содержимое веб-страницы, включая текст, изображения, ссылки, таблицы и другие элементы, которые будут отображаться на странице.
7) ```<script>``` and ```<link>```
Элементы ```<script>``` и ```<link>```* могут находиться как внутри ```<head>```, так и внутри ```<body>```. ```<script>``` используется для встраивания JavaScript кода, а ```<link>``` для подключения внешних файлов стилей и других ресурсов.

Пример: 
```HTML
<script src="script.js"></script> 
<link rel="stylesheet" type="text/css" href="styles.css">
```
### HTML: Ссылки
**Ссылки** в HTML - это **интерактивные** элементы, которые позволяют пользователям **переходить** с одной веб-страницы на другую, переходить между разделами одной страницы или выполнять другие действия, такие как отправка электронной почты.

>[!NOTE]
>Главный элемент, используемый для создания ссылок, - это тег ```<a>```

Для **создания** ссылки используйте тег ```<a>```, в который в атрибуте ```href``` указывается URL, куда будет вести ссылка.

**Пример**:
```HTML
<a href="https://www.example.com">Перейти на примерный сайт</a>
```
В данном случае, текст "Перейти на примерный сайт" становится кликабельной ссылкой, и при клике на нее пользователь перенаправляется на веб-сайт "https://www.example.com".

>[!WARNING]
> В ссылке без ```href``` лучше оставить подсказку о том, почему мы это сделали. Подсказку можно добавить с помощью атрибута ```title```. Подсказка появится, когда курсор задержится над ссылкой некоторое время.


#### Абсолютные и относительные ссылки
**Абсолютные** и **относительные** ссылки в HTML - это два основных способа указания URL (Uniform Resource Locator), куда будет вести гиперссылка. Они различаются в том, как **указывается** путь к ресурсу, на который ссылается гиперссылка.

**Абсолютные ссылки**
1) Абсолютные ссылки содержат полный веб-адрес (URL) ресурса, включая протокол (например, "http://" или "https://"), доменное имя (например, "www.example.com") и путь к ресурсу (например, "/page.html").
2) Эти ссылки указывают на **конкретный** ресурс вне зависимости от текущего расположения веб-страницы, где они находятся.
**Пример** абсолютной ссылки:
```HTML
<a href="https://www.example.com/page.html">Абсолютная ссылка</a>
```

**Относительные ссылки**
1) Относительные ссылки содержат путь к ресурсу **относительно** текущего расположения веб-страницы. Они **не** включают протокол и доменное имя, так как они определены в текущем контексте.
2) Эти ссылки позволяют ссылаться на ресурсы **внутри** того же сайта или внутри текущей папки, их расположение определяется относительно текущей страницы.

**Примеры** относительных ссылок:
- Ссылка на другую страницу в той же папке:
```HTML
<a href="page2.html">Относительная ссылка</a>
```
- Ссылка на изображение в подпапке:
```HTML
<img src="images/picture.jpg" alt="Относительная ссылка">
```

!!  **Преимущества и использование** !!
1) **Абсолютные** ссылки:
   - Используются, когда важно указать **конкретный** внешний ресурс.
   - Подходят для ссылок на **внешние** веб-сайты и ресурсы.
   - Гарантируют точность и **надежность** ссылок.
2) **Относительные** ссылки:
   - Чаще используются для ссылок **внутри** одного сайта или внутри одной папки.
   - Обеспечивают более гибкую и **удобную** навигацию внутри сайта.
   - Позволяют создавать **переносимые** сайты, где ссылки не зависят от изменения доменного имени или структуры сайта.

**Открытие в новом окне/вкладке**
Можно указать атрибут ```target="_blank"``` в теге ```<a>```, чтобы ссылка открывалась в новом окне или вкладке браузера. Это может быть полезно, чтобы сохранить текущую страницу пользователя от перезагрузки.
```HTML
<a href="https://www.example.com" target="_blank">Открыть в новой вкладке</a>
```
**Якори**
Ссылки могут также использоваться для **навигации** внутри одной страницы с помощью якорей. Якорь определяется с использованием атрибута ```id``` и ссылается на элемент на странице.
```HTML
<a href="#section2">
  Перейти к разделу 2
</a> 

<br><br><br><br><br><br><br><br><br>
<br><br><br><br><br><br><br><br><br>
<br><br><br><br><br><br><br><br><br>
<br><br><br><br><br><br><br><br><br>

<h2 id="section2">Раздел 2</h2>
```

**Ссылки на электронную почту**
Ссылки могут быть использованы для создания ссылок на адреса электронной почты. В атрибуте ```href``` указывается адрес с префиксом ```"mailto:"```.
```HTML
<a href="mailto:example@example.com">Написать почту</a>
```

**Ссылки на скачивание файла**
Для создания ссылки на скачивание файла в HTML, вы можете использовать тег ```<a>``` (anchor) с атрибутом ```href```, указывающим путь к файлу. Чтобы браузер понял, что файл должен быть загружен, а не открыт в браузере, вы можете добавить атрибут ```download``` в тег ```<a>```. В атрибуте ```download``` вы также можете указать желаемое имя файла, которое будет предложено пользователю при скачивании.

```HTML
<a href="document.pdf" download="Имя файла.pdf">Скачать документ</a>
```

Если хотим, чтобы файл не открывался в браузере, то мы должны указать в атрибуте ```download``` пустую строку.
```HTML
<a href="myFile.txt" download="">Скачать документ</a>
```

### HTML: Изображения
**HTML-изображения** позволяют вставлять **графические** элементы, такие как фотографии, иллюстрации и другие виды изображений, в веб-страницы. Для вставки изображений в HTML используется тег ```<img>```.

**Вставка изображений в теге ```<img>```**

1) Тег ```<img>```:
   - Элемент ```<img>``` используется для **вставки** изображений в веб-страницу.
   - **Главным** атрибутом этого тега является src, который указывает путь к изображению. Это может быть относительный или абсолютный URL к изображению.
   - **Пример** вставки изображения:
     ```HTML
     <img src="image.jpg" alt="Описание изображения">
     ```
2) **Атрибут** ```alt```:
   - Атрибут ```alt``` предназначен для предоставления **текстового описания** изображения. Этот текст будет отображаться, если изображение **не** может быть загружено, а также для доступности, чтобы пользователи с ограниченными возможностями могли понять смысл изображения.
   - Хорошей практикой является всегда включение атрибута ```alt```.
3) **Размеры изображения**:
   - Вы можете указать **размеры** изображения с помощью атрибутов ```width``` (ширина) и ```height``` (высота). Эти атрибуты определяют размеры изображения в пикселях
   - **Пример** установки размеров:
     ```HTML
     <img src="https://yt3.ggpht.com/ytc/AKedOLSydiH6sa2f6DtMrAnyViAZKj4jx56SJEjUTiwS=s900-c-k-c0x00ffffff-no-rj" alt="Описание изображения" width="300" height="300">
     ```

### HTML: Атрибуты стиля

**Атрибут style**
Данный атрибут может быть применен к любому тегу, однако только в рамках тела сайта. Значением этого атрибута выступают операторы тех стилей, которые необходимо применить к заданному элементу.

**Пример кода**:
```HTML
<body style="background-color:ivory;">
 
    <h1>Посмотрите на цвета и стили</h1>
    
    <p style="font-family:verdana;color:red;">
      Этот текст написан красным цветов с помощью шрифта Verdana.
    </p>
    
    <p style="font-family:times;color:green;">
      Этот текст написан зеленым цветом с помощью шрифта Times.
    </p>
    
    <p style="font-size:30px;">Этот текст имеет размер 30 пикселей.</p>
 
</body>
```

### HTML: Кнопки
Для создания кнопок используется тег ```<button>```. Внутри него размещается текст или изображение, которые будут отображаться на кнопке. **Например**:
```HTML
<button>Я кнопка</button>
```
>[!WARNING]
> Чтобы задать кнопке имя, тип или состояние, нужно добавить атрибуты: ```name, disabled и type```.
```HTML
<button type="button">Я кнопка</button>
<br><br>
<button type="button" name="i-button">Я кнопка</button>
<br><br>
<button type="button" disabled>Я заблокированная кнопка</button>
```
- `name` - имя кнопки.
- `disabled` - блокирует доступ к кнопке.
- `type` - тип кнопки.
- `submit` - задаётся кнопкам для отправки формы. Когда пользователь нажимает на кнопку с таким типом, браузер отправляет данные формы на сервер.
- `reset` - добавляется кнопкам сброса формы. Когда пользователь нажимает на такую кнопку, браузер возвращает значения всех полей формы к изначальным.

#### Кнопка с помощью тега `input`

Создать кнопку можно и с помощью тега `<input>`, если указать ему тип `button`:
```HTML
<input type="button" value="Я тоже кнопка">
```
>[!WARNING]
> У кнопок, созданных на инпуте, есть ограничения: сложно управлять размерами и положением изображений, а также нет псевдоэлементов. Поэтому оставьте `<input>` для создания элементов формы, таких как текстовые поля, радиокнопки или чекбоксы. А кнопки верстайте с помощью `<button>`.

#### Иконка на кнопке с помощью тега `<img>`
Способ подойдёт для **контентных** изображений.
```HTML
<button type="button">
  <img src="icon.svg" alt="Иконка" width="20" height="20">
</button>
```
>[!WARNING]
> Так вы можете добавлять кнопки с эмодзи, лайками, дизлайками или чем угодно. Иконке нужно обязательно задавать размеры, чтобы кнопка зарезервировала место и не прыгала, когда загрузится иконка.

### HTML: Формы
**HTML-формы** представляют собой важный элемент веб-страниц, который позволяет пользователям **взаимодействовать** с веб-сайтом, **отправлять** данные и **вводить** информацию.

#### Элемент `<form>`
1) Форма в HTML создается с использованием элемента `<form>`.
2) Элемент `<form>` определяет **контейнер**, в котором содержатся элементы управления (такие как текстовые поля, кнопки и флажки), которые пользователь будет использовать для ввода данных.

**Пример создания формы**:
```HTML
<form action="/submit" method="post"> 
    <!-- Здесь находятся элементы управления --> 
</form>
```

#### Атрибуты `action and method`
1) Атрибут `action` указывает URL, куда данные формы будут отправлены при ее отправке. Это может быть URL обработчика формы на сервере.
2) Атрибут `method` определяет метод, который будет использоваться для отправки данных. Два наиболее распространенных метода - "get" (данные передаются через URL) и "post" (данные передаются в теле запроса).

#### Различсные элементы управления
Формы могут содержать различные элементы управления, такие как текстовые поля, поля выбора, флажки, радиокнопки, кнопки отправки и другие.

Пример элементов управления:
```HTML
<input type="text" name="username" placeholder="Введите имя">
<select name="country">
  <option value="usa">Россия</option>
  <option value="canada">Беларусь</option>
</select>
<input type="checkbox" name="subscribe" value="yes">
```

#### Элемент `<label>`
Элемент `<label>` используется для создания текстовых меток, связанных с элементами управления в форме. Метки улучшают доступность и удобство использования формы.

**Пример** использования элемента `<label>`:
```HTML
<label for="username">Имя пользователя:</label>
<input type="text" id="username" name="username">
```
![image](https://github.com/rebys4/HTML_CSS/assets/145033292/863708e6-872b-483b-9abc-628d7497c375)

#### Кнопки отправки и сброса
1) Для **отправки** данных формы используется кнопка "Submit" (отправить), обозначаемая элементом `<input>` с атрибутом `type="submit"`.
2) Для очистки полей формы можно использовать кнопку "Reset" (сброс), обозначаемую элементом `<input>` с атрибутом `type="reset"`.

**Пример кннопок отправки и сброса**
```HTML
<form>
  <label for="username">Имя пользователя:</label>
  <input type="text" id="username" name="username">
  
  <input type="submit" value="Отправить"> 
  <input type="reset" value="Сбросить">
</form>
```

>[!WARNING]
> **Валидация форм**
> 1) HTML5 предоставляет встроенные механизмы валидации форм, такие как атрибуты `required` для обязательных полей и типовые атрибуты, такие как `pattern` для задания шаблонов данных.
> 2) Вы также можете использовать JavaScript для кастомной валидации форм.


#### Элемент `<input>`
Элемент `<input>` позволяет пользователям вводить и выбирать различные типы данных, такие как текст, числа, даты и многое другое. Вот основные аспекты элемента `<input>`:
1) **Типы** `<input>`:
   - `text`: для ввода однстрочного текста
   - `password`: для ввода пароля
   - `email`: для ввода адреса эл. почты
   - `number`: для ввода чисел
   - `date`: для выбора даты
   - `checkbox`: для флажков
   - `radio`: для радиокнопок
   - `file`: для загрузки файлов с компа
   - и многие другие....
**Примеры**
```HTML
<input type="text" name="username"><br><br>
<input type="password" name="password"><br><br>
<input type="email" name="email"><br><br>
<input type="number" name="quantity"><br><br>
<input type="date" name="birthdate"><br><br>
<input type="checkbox" name="subscribe" value="yes"><br><br>
<input type="radio" name="gender" value="male">
<input type="radio" name="gender" value="male"><br><br>
<input type="file" name="uploadFile">
```
![image](https://github.com/rebys4/HTML_CSS/assets/145033292/b9245991-c56b-45a9-a510-e3a7afcc04f8)
2) **Атрибуты** `name` и `value`:
    - Атрибут `name` используется для **идентификации** поля в форме, когда данные отправляются на сервер. Каждое поле формы должно иметь уникальное имя.
    - Атрибут `value` определяет начальное значение поля, которое может быть предварительно заполнено.

**Пример** использования `name` and `value`:
```HTML
<input type="text" name="username" value="John">
```
3) **Атрибут** `placeholder`:
   - Атрибут `placeholder` позволяет указать **текст-подсказку**, которая отображается внутри поля ввода и предостерегает пользователя о том, что ожидается ввод.

**Пример** использования `placeholder`:
```HTML
<input type="text" name="username" placeholder="Введите ваше имя">
```
4) **Атрибут** `required`:
   - Атрибут `required` указывает, что поле обязательно для заполнения перед отправкой формы. Если оно не заполнено, браузер выдаст сообщение об ошибке.

**Пример** использования `required`:
```HTML
<input type="email" name="email" required>
```

5) **Атрибут** `disabled`:
   - Атрибут `disabled` делает поле недоступным для ввода и отображает его в сером цвете. Это может быть полезно для временного отключения полей в зависимости от контекста.

**Пример** использования `disabled`:
```HTML
<input type="text" name="username" value="John" disabled>
```

6) **Атрибут** `readonly`:
   - Атрибут `readonly` предотвращает редактирование поля, но позволяет просматривать его содержимое.

**Пример** использования `readonly`:
```HTML
<input type="text" name="username" value="John" readonly>
```
